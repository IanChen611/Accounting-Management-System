## 發票編號：
- [ ] 因為匯出時，報表的排序會以日期為優先排序，如果無法更改排序的欄位，希望可以改成三家公司各兩本發票，可能編成A1.A2.B1.B2.C1.C2之類的。
- [ ] 篩選後，匯出報表，會把所有在資料庫的全部匯出，並不是只匯出篩選過後的，這樣將來資料庫的資料一多，會很可怕 => 我目前看到的是可以只匯出篩選後的資料

##  日期部分：
- [x] 希望可以直接輸入1101或1102就可以跳出2025-11-01(但希望2025可以用114表示)
- [x] 日期輸入完案ENTER可以直接跳下一欄位
- [ ] 作廢發票可以不要輸入日期 => 目前因為需要照日期排序，所以日期沒辦法不輸入

## 客戶代號部分：
- [x] 輸入代號ENTER就可以跳出簡稱，不要再用滑鼠點
- [x] 客戶代號可以直接有下拉搜尋的方式

## 數量與金額方面：
- [x] 數字希望可以有千分位的符號(，) 我要讀數字比較快速
- [x] 數量欄位預設空白自己輸入就好，不要預設1
- [x] 金額欄未預設空白自己輸入就好，不要預設0
